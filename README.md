# Lilly Lens Photography
Final Project for Web Design 2

## Creator 
Kelsey Adams 

## Description 
This project focuses on creating a modern website for a photographer whose work centers around professional headshots, while also highlighting a range of event and lifestyle photography. The current online presence is good but could improve structure wise, and it also does not have a working phone layout.

The updated site will be clean and sheek to really put focus on her work. Ill do this by using strong images to immediately capture the viewers attention. Visitors will be able to easily explore different types of sessions ranging from engagements, graduations, and special events without feeling overwhelmed. The overall goal is to present a polished brand, make it simple for potential clients to reach out and book!

## Target Audience 
Primary - Professionals, students, and creatives who need clean, high quality headshots for resumes, portfolios, LinkedIn, or personal branding. 

Secondary - Individuals and couples planning milestone moments like engagements, graduations, baby showers, or celebrations. 

## Project Goals 
-Build a visually engaging website that reflects both professionalism and personality.
-Organize portfolio work into clear sections so visitors can quickly find the type of photography they need.
-Make contacting or booking simple 
-Provide helpful details about sessions (what to expect, pricing, or packages)
